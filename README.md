# 🎨 Gerador de Cores Dinamico

Uma aplicação simples feita para treinar os conceitos básicos de JavaScript e de alternancia de styles de um CSS.

## O que o projeto apresenta?

O estudo feito mostra a alternância de um estilo css para outro através de um códigop via javascript.

## Principais aprendizados

* Criar variáveis em JS
* Criar array para gerador simples
* Criar um botão funcional
* Utilizar o método de math.random para randomizar as cores.
* Alternância de páginas
* Estudar Hexadecimal para desenvolver o projeto
* Randomização de hexadecimais em forma de arrays

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Conceitos aplicados

* Manipulação do DOM
* Eventos com addEventListener
* Criação dinâmica de elementos
* Arrays

## Objetivo

O objetivo do projeto foi entender como funciona o JavaScript e explorar a ferramenta para futuros usos.
